# Search-Engine  
**Flexi - Credit Mini Project**  

### Setup Instructions  

1. **Extract Contents:**  
   Unzip the project files.

2. **Backend Setup:**  
   - Navigate to `login/backend`.  
   - Run:  
     ```bash
     npm install  
     npm run  
     ```

3. **Frontend Setup:**  
   - Navigate to `login/frontend`.  
   - Run:  
     ```bash
     npm install  
     npm run  
     ```

4. **Database Setup:**  
   - Create a MongoDB database named `Search-engine`.

5. **Run the Application:**  
   - Execute the Python script:  
     ```bash
     python app.py  
     ```
